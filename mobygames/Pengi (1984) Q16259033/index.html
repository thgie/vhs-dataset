Description

Pengi is a clone of Pengo. It is a single-screen game where Pengi, the penguin, is trapped in a maze of ice blocks and surrounded by an electric fence. There are also enemies, called Snobees, that spawn regularly. Ice blocks can be pushed by moving into them, which causes them to travel across the screen until they reach an obstacle. Points are scored by connecting diamonds in the maze and by pushing blocks into the Snobees to freeze them, after which they can be collected. Snobees are able to destroy blocks in their way. The environment is randomized for each session, there are three lives at the start and the game keeps track of the score.
